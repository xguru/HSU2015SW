구글 검색이 좋은이유

우리는 정보화 시대에 살고 있다. 정보화시대에서 정보는 자산이 된다.
사람들은 궁금한 정보가 있을때 검색엔진을 사용한다.
우리가 쓰는 검색엔진의 종류에는 여러가지가 있지만 그 중 최고의 검색 엔진은 구글이다.
구글은 최초의 검색엔진은 아니지만 최고라고 할 수 있다. 
구글 검색엔진이 좋은 이유에는 여러가지가 있다.

1. 페이지랭크 알고리즘
    - 구글은 페이지랭크 알고리즘을 사용한다. 이는 사용자가 검색을 했을때 랭크가 높은 페이지 
      즉 신뢰성 높은 페이지를 검색 결과 위로 올려 사용자가 원하는 정보에 접근하는 것을 도운다. 
      
2. 단순한 인터페이스 
    - 구글의 메인 페이지는 다른 검색엔진과 다르게 단순한 인터페이스를 지니고 있다. 
      이는 광고영상과 실시간검색어와 같은 사용자가 원하는 검색과 상관없이 보여지는 
      것들이 로드되는 시간을 생략하며 직관적인 인터페이스로 원하는 기능에 대한 빠른 실행을 도운다.

3. 자료가 많다.(영어 자료)
    - 이것의 예시로 보통 개발을 할 때 에러가 나는 부분을 구글에서 해당 오류를 영어로 검색하면 나와 같은 증상을 지닌
      사람들의 해결방안을 쉽게 찾을 수 있다. 반면 국내의 네이버의 경우 한국문서만 나올 뿐 원하는 자료를 
      찾기 힘들다.  

4. 광고를 우선시 하지 않는다.
    - 구글은 결과를 검색했을때 스폰서링크 같은 광고사이트 보다 자신이 검색한 검색 결과를 보여준다.
      이를 통해 사용자는 자신에게 필요없는 광고를 보지않고 자신이 찾는 정보에 쉽게 접근이 가능하다.
      이런 결과로 사용자는 구글검색에 대한 신뢰성이 올라간다.

5. 파일 타입 검색
    - 구글은 검색시 pdf파일이나 ppt 와 같은 파일 타입을 입력하고 검색시에 파일에 대한 정보가 나온다.
      이는 우리가 자료를 찾을 시에 많은 도움이 된다. 사용방법은 filetype:pdf(검색 할 파일타입) 구글(검색할 단어)
      과 같이 간단하게 파일을 찾을 수 있다.
 
6. 검색엔진 api
    - 구글 검색엔진은 api가 존재하여 이를 개발자가 사용할 수 있다. 이를 통해 자신의 홈페이지내의 문서 및 글을
      검색하여 사용하는 것이 가능하다. 이외에도 구글은 많은 api 를 무료로 제공하고 있다.

7. 구글의 마인드
    - 이러한 장점들은 이용자들을 우선시 하는 구글의 마인드에서 나온다고 생각해도 과언이 아니다.
      위의 예시 중 광고, api 공개, 단순한 인터페이스 같은 것은 모두 이용자를 중심으로 생각했기에 
      가능한 발상이라고 생각한다.
