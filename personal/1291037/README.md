* 아이디어 구상
대중교통 목적지 도착을 알려주는 어플리케이션으로 사용자가 이용할 대중교통과 목적지를 설정한다.
후에 사용자가 도착 전 몇 정거장을 설정한 후 그 목적지에 도달하면 푸시를 통해 알림을 준다. 
 
* 만들 소프트웨어가 사용자에게 주는 가치 3가지
- 출퇴근 시간에 내릴 장소를 지나칠 걱정 없이 편하게 있을 수 있다.
- 사용자가 목적지를 선택함으로써 해당 시간의 혼잡정도등 통계를 낼 수 있다.
- 한개의 어플리케이션으로 버스 어플리케이션과 지하철 어플리케이션의 기능을 통합하여 활용할 수 있다.
 
* 만들 소프트웨어의 주요 기능 3가지
- 사용자가 목적지를 설정한 후 목적지에 도달하면 푸시를 준다.
- 버스와 지하철의 현재 위치를 실시간으로 확인가능하다.
- 어플리케이션에서 현재 목적지로 향하는 인구 수 등 통계를 그래프로 쉽게 확인가능하다.
