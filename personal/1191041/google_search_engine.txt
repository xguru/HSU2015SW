구글의 검색이 뛰어난 이유

구글의 검색이 뛰어난 이유는 페이지 랭크라는 알고리즘을 쓰고 있기 때문이다.
페이지랭크란 www와 하이퍼링크 구조와 같이 문서에 상대적으로 중요도를 가지는 문서에 더욱
가중치를 부여 하는것이다. 

간단히 설명하면 페이지 a가 b c d라는 총 3개의 페이지에 링크를 걸면 b는 a페이지의 3/1랭크값을 가져오는것이다.
이것을 "정규화"방식이라고 하는데, 페이지 a는 a라는 페이지를 인용하고 있는 b c d 페이지가 가진 
페이지 랭크를 "정규화"한 값의 합이다.
이러한 정규화 방식은 예를 들면  페이지 a를 가리키는 페이지가  b c d가 있으면 b가 가지고 있는
페이지 랭크값이 0.1이고 c가 가지고 있는 랭크값이 0.4이고 ,d 가 가지고 있는 페이지랭크값이 0.3이면
페이지 a의 페이지 랭크값은 0.8이 되는것이다. 나머지 0.2의 페이지 랭크값은 또다른 페이지 a를 인용
하고 있는 페이지에서 페이지랭크값을 쓰고 있는것이다. 또한 b 가 0.1이 었으면 나머지 0.9도 또다른
페이지에서 페이지 랭크값을 쓰고 있는것이 된다.


페이지랭크 알고리즘
PR(a) = (1-d)/N(전체 페이지수) + d (PR(b)/C(a) + … + PR(an)/C(an)) 가 있는데
PR(a)는 a가 가지고 있는 페이지 랭크값이고 d는 여기서 어떤 웹서핑을 하는 사람이 다른페이지로 가는 확률이라고 한다.
d가 커질수록 계속 다른 페이지를 링크하는것이고 d가 작아지면 그 페이지에 만족하여 오랫동안 그 페이지를 방문하고 있는것이다.
페이지 랭크값을 책정하는것은 PR(an)/C(an) ,즉 a라는 페이지 링크에 a페이지가 가지고 있는 링크의 총갯수를 나눈것을 의미한다.
이렇게 하면 전체 페이지의 PageRank를 합산한 값이 1이 된다.
