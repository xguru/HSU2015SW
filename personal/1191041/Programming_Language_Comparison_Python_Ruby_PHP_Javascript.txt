PHYTHON의 장점

여지껏 배웠던 c언어와 c++, java와는 달리 파이썬을 써본후 확실히 느낀 첫번째 장점은 굉장히 간결(?) 깔끔하게 작성할 수 있다는것이다.
자바나 씨언어등 코딩을 할때 { } 를 많이 쓰는데 파이썬은 이부분이 없어 훨씬 전체 코드를 볼때 읽기가 좋은것 같다.
phython에서도 함수구현부분이 비슷하여 큰 혼동이 일어나지 않았다.예를들어 매개변수나 리턴도 똑같다.하지만 편하게 작성할수 있었던 함수들도 있어 
max min등은 괜찮은 생각같았다.
매번 코딩을 할때면 char  int string 이렇게 타입을 적고 변수를 써야 했지만 바로 변수선언에 값을 주면 되어 굉장히 편하다는 장점이 있다.
그리고 또한 가장 매력적인것은 씨언어 공부할대 한창 애를 먹었던 포인터 개념이 없어서 좋았다.
list를 학습할때도 정말 편하다고 느낀것은 다른 언어 배열에서는 가장 기본적인 홀수 짝수 또는 덧셈 뺄셈 프로그램도 몇줄정도는 코드를
적어야 하는 번거로움이 있었지만 list를 이용하여 정말 쉽고 빠르게 코딩을 할수 있었다.
마지막으로 장점이기라기 보다는 새로운 방법으로 입출력을 할수 해봤던것 같다. 파일을 불러서 원하는 출력값을 코딩해 보았는데
이것또한 굉장히 쉽게 접근할수 있어 어려움이 없었다.

RUBY의 장점

루비는 코드 자체를 매우 간결하게 짤수 있어 쉽고 빠르게 습득할 수 있었다. 무엇보다 루비는 훨씬 파이썬보다도 쉬운 언어라는것이다.
앞서 배웠던 파이썬과 같이 변수선언과 {}을 생략하는것도 아주 유사하다. 문자를 숫자로 숫자를 문자로 바꾸어 서로 합하는 코딩을 하였는데
이것을 다른언어에서 사용한다면 갖가지 함수와 많은줄을 들여가며 코딩을 해야 했지만 정말 너무나도 짧은 코드이다. 
루비는 또한 메소드가 비교적 쉽게 다가온다. 다른언어들과는 달리 조금이라도 더 외울(?)수 있는 쉬운 메소드들이 있다.
그전에 변수를 선언하여 그 변수에 바로  .을 찍어 메소드와 연결하는것은 정말 괜찮고 좋은방법인것을 느꼇다.
.each do 를 사용하여 배열의 모든 영역에 간결히 접근하는것은 놀라웠다.이러한 것들이 가능한 이유가 바로 block 이었다.
생소해보이지만 효과는 정말 강하다는것을 느꼇다.포인터의 개념이랑 비슷한데 배열을 이용한 최고의 접근법이라고 생각한다.
proc또한 block과 비슷하지만 가장 큰 강점은 함수에 인자를 넣어서 호출할수 있다. 이것으로 인해 새로운 생각을 많이 할수 있을것 같다.

 
PHP의 장점

웹에 특성화된 언어이지만 PHP는 문법적으로 C와 많은 유사성이 있어 웹페이지 개발에 두려움없이 쉽게 접근할수 있다는 가장큰 장점이
있는것 같다. 자주 사용하는 조건문과 반복문, 기본 구조 또한 c언어와 아주 유사하여 초보자들도 구현이 금방이라는것을 느꼈다.
PHP에서의 큰 장점은 아무래도 함수를 이용하여 코드를 작성하는것이다. String 함수와 Math함수 array함수들을 PHP에서 제공하는 함수로
다양한 조건들을 금방 구현할수 있었다. 특히 배열함수에서 sort와 rsort를 이용하여 큰수와 정렬을 시키는것은 PHP 언어만의 특별한 경험을 
하였다.PHP는 C의 구조체와 C++의 클래스를 이용하여 코드를 작성하여 페이지상에서의 다양한 접근을 할수 있었다.C언어만의 특징과 웹과의 
만남으로 웬만한 c학습자는 누구나 가능할것이라고 생각한다. 다행이라고 생각하는것은 main이나 다른 함수선언문이 떨어져있는것이 아니라
바로 밑에다 코드를 구현할수 있어 속도감을 증가시킨것 같다.

JAVA SCRIPT의 장점

자바 스크립트는 코드자체가 매우 간결하고 깔끔하다. 스크립트라는 말 그대로 각본처럼 간략히 적어 코딩을 하는것이 느껴진다.
자바를 공부한 사람이라면 출력문과 입력문을 제외하고는 대부분 유사하여 금방 따라 할수 있을것 같다.한가지 아주 유용한것은
변수를 선언할때 type을 모두 var로 시작하고 변수 선언을 제외하고도 구조체에서도 쓰이고 자바스크립트를 사용하는
사용자들에게 확실한 편리는 주는것 같다..자바 스크립트는 앞서 본 PHP와 유사하게 HTML문 즉 웹에 특화 되어 있어
실행을 할 환경을 만들지 않아도 된다.자바 스크립트의 가장 큰 묘미는 함수를 이용한 코드작성인것 같다. 구조체 안에 변수를 넣고
함수를 선언하여 구조체 바깥쪽에서 구조체변수를 호출하여 그 안에서 실행하는것은 굉장히 코드를 빠르게 작성하는데 도움을 많이
줄것 같다.또한  자바 스크립트도 따로 main없이 바로 작성 할수 있어 코드작성의 속도를 더욱 향상시킬수 있다.
